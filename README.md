## 项目描述

该项目使用三层架构编写，通过反射对Servlet程序进行封装、对数据的操作用BeanUtils工具类进行了封装、文件上传、封装分页等，用到的技术点有：Servlet程序、反射、jsp页面、EL表达式、JSTL标签库、Session会话、JSON使用、Ajax请求等。项目的功能比较简单，比较合适拿来学习，所有界面来源于github，如果想完善这个项目的可以进行二开。
